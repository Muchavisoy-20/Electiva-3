# DevLocker API

API REST desarrollada con **Node.js, Express y TypeScript** para gestionar snippets de código con autenticación JWT.

# Tecnologías usadas

- Node.js
- Express
- TypeScript
- MongoDB + Mongoose
- JWT (Autenticación)
- Bcrypt
- Dotenv


#  Requisitos previos

Antes de ejecutar el proyecto asegúrate de tener instalado:

- Node.js (v18 o superior)
- MongoDB (local o Atlas)
- npm o yarn
